# vue-covid-tracker

Приложение для изучения js-фреймфорка Vue. 

Ковид-трекер, показывающий количество заражений и смертей в разных странах. Данные берутся из открытого API.

### Используемые технологии:
- TypeScript
- Vue
- Vite

### Ключевые навыки:
- Работа с API
- Адаптивная верстка
- Работа с компонентами Vue
